# Sales-Performance-Dashboard
This project is an interactive Excel dashboard designed to track and analyze sales performamce across various regions. it provides business insights into product trends, regiional dominance and revenue forecasting.

# Key Features
1. Interactive slicers: these filter data by Region and Product Category.
2. Trend analysis: monthly revenue tracking to idetify any patterns.
3. Top Performers: visual breakdown of the top 5 products by total revenue.
4. Regional Insights: comparison of sales across the regions.

# Tools and Skills used
1. Excel Functions: data cleaning, 'TEXT()' for month formatting and calculated columns.
2. Pivot Tables & Charts: data aggregation and visualization
3. Dashboard Design: UI/UX principles (removing guidelines, consistent color palettes and alignment. This was doen for uniformality)

# Insights from the Data
1. Regional Hub: Gauteng consistently leads in revenue suggesting a higher concentration of corporate clients
2. Product Focus: High Value electronics electronics like Laptops drive more revenue than high volume furniture items.
3. Growth: Q1 showed a steady upward trend in sales across all categories
